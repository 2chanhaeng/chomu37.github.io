## 반응형 CSS

### CSS 단위

- px: 픽셀 수
- %: 부모 요소의 비율
- em: 자기 자신의 폰트 사이즈
- rem: root 요소의 폰트 사이즈
- vw: viewport의 너비
- vh: viewport의 높이

### 변수

- CSS 변수 선언

```css
:root {
  --변수명: 값;
}
```

- CSS 변수 사용

```css
선택자 {
  속성: var(--변수명);
}
```


### 미디어 쿼리

```css
@media (조건) {
  선택자 {
    속성: 값;
  }
}
```
